Temperature Conversion

This ruby program will prompt the user for a temperature in degrees Celsius and let the user know what the corresponding temperature is in Fahrenheit. 

def celsius_to_fahrenheit(celcius)
  return (celcius * 9 / 5) + 32
end

puts "Please indicate the temperature in Celsius you wish to convert: "
celcius = gets.to_i
puts celsius_to_fahrenheit(celcius)

def suffix(user_input)
  if (11..13).include?(user_input % 100)
    return "th"
  elsif last_digit(user_input) == 1
    return "st"
  elsif last_digit(user_input) == 2
    return "nd"
  elsif last_digit(user_input) == 3
    return "rd"
  else
    return "th"
  end
end

def last_digit(user_input)
  return user_input % 10
end

puts "Please enter a number: "
user_input = gets.chomp.to_i
puts "That's the #{user_input}#{suffix(user_input)} item!"

Foobar

This ruby program is an adaptation of a classic technical interview problem which displays sequences of the Foobar pattern. 

def div_by_3(n)
  return n % 3 == 0
end

def div_by_5(n)
  return n % 5 == 0
end

current = 0
puts "How many items do you want to see?"
stop = gets.chomp.to_i

while current < stop
  current += 1
  if div_by_5(current) && div_by_3(current)
    puts "Foobar"
  elsif div_by_3(current)
    puts "Foo"
  elsif div_by_5(current)
    puts "Bar"
  else
    puts current
  end
end
